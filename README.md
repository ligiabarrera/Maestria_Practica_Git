# DataPrepUtils

DataPrepUtils es un paquete de Python diseñado para realizar tareas básicas de preparación de datos en proyectos de Ciencia de Datos.

## Estructura del proyecto

DATAPREP_PROJECT
│
├── .venv/                 # Entorno virtual
├── datapreputils/         # Paquete de utilitarios
│   ├── __pycache__/
│   ├── __init__.py
│   ├── cleaning.py
│   ├── normalization.py
│   └── statistics.py
├── main.py                # Archivo principal
├── README.md              # Documentación
└── requirements.txt       # Dependencias del proyecto

## Funcionalidades

### Módulo cleaning.py (Deisy Sánchez S.)
- Eliminar valores nulos
- Reemplazar valores faltantes

### Módulo normalization.py
- Normalización Min-Max
- Estandarización Z-score

### Módulo statistics.py
- Media aritmética
- Moda
- Media armónica
- Media geométrica

## Ejecución

1. Ubicarse en la carpeta del proyecto.
2. Ejecutar el archivo principal:

python main.py
